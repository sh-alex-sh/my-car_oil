## 1. 项目概述

**一句话描述：** 个人车主使用的油耗统计应用，纯本地存储，支持数据导出，支持 Android APK 侧载安装。

**目标用户：** 个人车主（1-2 辆车），追求简单易用。

**核心价值：** 记录加油数据 → 自动计算油耗 → 了解车辆花费 → 数据随时导出。

**已实现平台：**
- **Web（PWA）** — 浏览器访问，可添加到主屏幕
- **Android APK** — Capacitor 原生打包，华为等无 GMS 设备可侧载安装

## 2. 技术方案

| 维度 | 决策 |
|---|---|
| 平台 | Web PWA + Android APK（Capacitor） |
| 框架 | React 18 |
| 构建工具 | Vite 5 |
| 样式 | Tailwind CSS 3 |
| 状态管理 | Zustand |
| 本地数据库 | Dexie.js（IndexedDB 封装） |

| 路由 | React Router 6（Hash 路由） |
| PWA | vite-plugin-pwa + Service Worker |
| Android 打包 | Capacitor + Gradle |

## 4. 数据模型

### 4.1 加油记录表 `fuel_records` ✅

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | number (自增) | 主键 |
| `vehicleId` | number | 车辆 ID |
| `date` | string (ISO 8601) | 加油日期 |
| `mileage` | number | 当前里程表读数（公里） |
| `fuelAmount` | number | 加油量（升） |
| `fuelCost` | number | 加油总金额（元） |
| `fuelPrice` | number | 单价（元/升），自动计算 = fuelCost / fuelAmount |
| `fuelGrade` | string | 油品标号（92# / 95# / 98# / 0#） |
| `isFullTank` | boolean | 是否加满 |
| `note` | string | 备注（加油站名等） |
| `createdAt` | string (ISO 8601) | 创建时间 |

**油耗计算公式（加满法）：**
```
油耗 (L/100km) = 本次加油量 / (本次里程 - 上次里程) × 100
```
仅当 `isFullTank = true` 且存在前一条加满记录时才计算油耗。

**数据库索引：** `++id, vehicleId, date, mileage, fuelGrade`

### 4.2 车辆表 `vehicles` ✅ (提前至 Phase 1)

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | number (自增) | 主键 |
| `name` | string | 车辆昵称 |
| `brand` | string | 品牌 |
| `model` | string | 型号 |
| `plateNumber` | string | 车牌号 |
| `isActive` | boolean | 当前活跃车辆 |

### 4.3 后续表 (Phase 3)

- 保养提醒表 `maintenance_reminders`
- 油价记录表 `fuel_prices`
- 费用分类表 `expense_categories` / `expenses`
